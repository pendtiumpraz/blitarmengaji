Pilih titik dakwah untuk laporan terpisah:

Saldo Global = penjumlahan seluruh saldo titik dakwah aktif se-Blitar.

Kontrol atas: toggle Global/Per Titik + periode + Unduh PDF · ribbon ayat amanah (An-Nisa 58) · 4 KPI (saldo, masuk, keluar, surplus) · grafik batang 2-warna + donut conic-gradient · tabel transaksi + filter + pagination · panel arsip PDF + pemilih lingkup per-titik.
