2. Builder Menu Dinamis

Item di sini yang menentukan apa yang muncul di sidebar tiap role. Atur urutan (drag), permission yang dibutuhkan, nesting, dan aktif/nonaktif.

Sidebar admin dirender dari tabel menu_items + permission user. Item tanpa permission cocok = disembunyikan otomatis.
