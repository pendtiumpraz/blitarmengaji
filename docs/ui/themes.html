Cara kerja di aplikasi nanti

  • Per-user: tiap jamaah pilih tema favorit di Akun → Tampilan. Disimpan di users.theme_pref.
  • Default global: admin set tema bawaan platform di Pengaturan (settings.default_theme), butuh permission theme.manage.
  • Implementasi: data-theme di <html> + CSS variables. SSR baca cookie/preferensi → tanpa "kedip" saat load.
  • Tambah tema = murah: cukup 1 blok variabel baru. Bisa juga tema custom tersimpan di tabel ui_themes (tokens JSON) untuk admin bikin tema sendiri.
  • Dark mode = salah satu tema (Senja), plus opsi "ikuti sistem".

Catatan: warna brand resmi (Teduh: Hijau+Emas) tetap jadi default & identitas utama. Tema lain = opsi personalisasi "biar gak bosen".
